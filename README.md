# gridtool ver 1.0.1
960 Grid Generator for Photoshop CC

## Install: 
- Copy to folder: C:\Program Files\Adobe\<Adobe Photoshop folder>\Presets\Scripts
	
## Usage:
- Goto menu File > Scripts > Grid Template Generator...
